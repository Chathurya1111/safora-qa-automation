# Safora QA Automation Project

## Overview
This project automates the Contact Us form on the Safora website using Playwright.

## Tools Used
- Playwright
- Node.js

## Test Scenario
- Navigate to Safora website
- Open Contact Us page
- Validate navigation flow

## How to Run

1. Install dependencies:
npm install

2. Install browsers:
npx playwright install

3. Run test:
npx playwright test
